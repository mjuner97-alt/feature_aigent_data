# 技能名称
数据查询工具索引

## 功能描述
本技能用于帮助用户了解系统中有哪些工具可用，以及如何通过 router_tool 调用这些工具。支持：
1. 查看所有（除测试检查相关）数据可用工具及其功能简介
2. 通过 toolId 调用具体工具
3. 提供统一的 JSON 调用入口，简化工具调用流程
4. 支持参数类型自动转换，提高调用灵活性

## 使用方法
### 步骤1:选择目标工具

**缺陷密度**

|工具ID	                |功能描述|
|-----------------------|-----------------------|
|densityDeptCheck	    |部门（版本计划维度）缺陷密度查询工具，根据部门和版本计划字段查询缺陷密度查询工具数据|
